
## Getting Started

### Installation

1. Install NPM packages
   ```sh
   npm intall
   ```
2. Install Mongo Replication
   ```sh
   npm install run-rs -g
   ```

### Start Mongo Replication

  ```sh
   run-rs -v 4.0.0 --shell -h localhost
   ```


### Run API

   ```sh
    npm run start:nodemon
   ```

#### import data
     
  ```sh
   in postman http://localhost:3000/import-data
   ```
